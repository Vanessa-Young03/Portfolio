Book Rental Website

Topics Covered:

  • View
  • Insert
  • Search

On the View page, it will show the various of books that is available in the database, which include title, category, pages, quantity and rent price

On the Insert page, it will require the user to enter the details such as title, category, pages, quantity and rent price. After the user click on submit, it will insert into the database and show it on the view page.

On the Search page, It will require the user to enter the words that is contains in the book title. After the user click on search, the books that contains the words that the user search will displayed on the page.

Items Website

Topics Covered:

  • Edit
  • Delete

On the View page, it displays the list of items that can be edit and delete, when you click on the item name, it will show you the details of the items such as item name, description, date sold, quanlity and price

On the Edit page, it will automatically show the details of items that the user wanted to edit. After finishing editing and the user click on submit, the details will be updated in the database and show on the item list page.

On the Delete page, it will automatically show the details of items that the user wanted to delete. After the user click on delete, it will delete from the database and it will not show on the item list page.

Colour Changer Website

Topics Covered:

  • Session
  • Cookies

The user is required to register an account if they do not have.

The user is required to login if they already have an account.

After the user has login, it will direct them to the home page whereby they can choose the colour for their background.

After the user has choose the color and click on confirm, the backgroud will change the color that the user has choosen.

By using session, it can check if the user has logged in to access the protected pages.

By using cookie, it can set the duration of how long the color will stay in the background before it turn back to its original color which is white. Also, even the user has logged out and logged in again, the color will still remain the same as before when the user logged out.

Used Items Website

Topics Covered:

  • User rights and Management - defining user roles, assigning rights to roles, implementing autorization.

On the home page, it show the used item name, description, price and the owner name.

If the user login as a role of an admin, it is given the rights to delete everybody post including themselves.

If the user login as a role of an member, it is given the rights to edit their post only. They are not allow to edit other posts.

Movie Review Website

Topics Covered:

  • View
  • Insert
  • Update
  • Delete
  • Search
  • User rights and Management

The user will need to register with their details such as fullname, email address, date of birth, username and password.

The user can login in with their username and password to add, edit and delete reviews. If they did not login, the only can view movie information and review plus search movie

After the user has log in, it will direct them to the page whereby it will shows a list of movies. When they click on see more, it will show them the movie information such as movie genre, language, director, casts, synopsis

After the user click on reviews, it will direct them to the page whereby it require the user to enter their reviews and ratings for the movie. Once the review has been added, it will insert to the database and display on the review page.

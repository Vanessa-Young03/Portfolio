Online Reservation

Topics Covered:

  • Date Picker
  • Time Picker
  • Checkbox
  • Toast

The app is to allow customers to book a reservation for a restaurant.

The app requires the customers to enter details such as name, mobile number and size of the group. Plus, the date and time of the booking.The user can also select table in smoking or non-smoking area.

After the user click on the submit, a message contains all reservation information should be displayed on the screen for a short while by using toast message.

When the user click on reset, it will clear all the inputs , including resetting the time and date to default values.

Simple To Do

Topics Covered:

  • List View
  • Spinner

When “Add a new task” is selected, a hint “Type in a new task here” is showing on the input field. The button “DELETE” is disabled. The user can add the new task into the list by clicking on the “ADD” button

When “Remove a task” is selected, A hint “Type in the index of the task to be removed” is showing on the input field. The button “ADD” is disabled. The user can delete a task by filling in the index number and clicking on the “DELETE” button.

If no task is in the list, display a message "You don't have any task to remove" on the screen, rather than system crash

If the task doesn’t exist according to the index number the user entered, display a message "Wrong index number" on the screen, rather than system crash.

My Local Banks

Topics Covered:

  • Context Menu
  • Options Menu

By using the Context Menu, the user can either visit the website or contact the bank.

When the user click on website, it will direct the user to the website of the bank. When the user click on contact the bank, it will direct the user to the dialpad with the contact numbers auto fill-up.

By using the Option Menu, the user can either translate the app to english and chinese.

The user needs to click on the three dots on the top right hand conner to select their language preferences and it will display the name of the bank in different language accordingly.

Dialog Box

Topics Covered:

  • Simple Dialog
  • Button Dialog
  • Text Input Dialog
  • Date Picker Dialog
  • Time Picker Dialog

When the user click on the demo 1 button, it will pop up a dialog with the title "Congualations" plus the message "You have completed a Simple Dialog Box".

When the user click on the demo 2 button, it will pop up a dialog which requires the user to choose yes, no or cancel. If the user click on yes, "You have selected positive" text will appear next to the button.If the user click on no, "You have selected negative" text will appear next to the button. If the user click on cancel, no text will appear next to the button.

When the user click on the demo 3 button, it will pop up a dialog which requires the user to enter two numbers. After the user click on OK, the sum of the two numbers will appear next to the box.

When the user click on the demo 4 button, it will pop up a dialog which requires the user to select the date on the date picker. After the user click on OK, the date that the user select will appear next to the box.

When the user click on the demo 5 button, it will pop up a dialog which requires the user to select the time on the time picker. After the user click on OK, the time that the user select will appear next to the box.
